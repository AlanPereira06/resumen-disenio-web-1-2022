HTML

Navegá el arbol HTML

Toca boton derecho sobre el elemento que estas interesado y selecciona "Inspeccionar".

Se abrirá la consola de desarrollo en la tab "Elementos". Ahi vas a poder ver en cual elemento esta contenido (llamado padre) y cuales elementos contiene (llamado hijos).

Seleccionando el elemento en el arbol vas a poder ver las reglas css (y otras cosas). Las podes cambiar ahi, pero al refrescar la pagina se van a borrar.

En esta consola de desarrollo podemos inspeccionar y modificar TODA la pagina.

CSS

Agrega estilos a tu página

Utilizamos CSS (cascade style sheets) para darle el diseño que queremos a nuestra pagina.

La ventaja de CSS es que permite separar el contenido (HTML) del diseño (CSS), logrando un código más ordenado.

CSS

Método 1: CSS inline

La forma mas rudimentaria de agregar estilos a un elemento es, en la mismo elemento HTML (En la etiqueta de apertura) usar el atributo "style".

Todos los elementos aceptan esta propiedad.

Este método tiene la desventaja que mezclamos los estilos con el contenido. Esta bien para prototipar (probar cosas) pero no para un producto comercial.

CSS

Método 2: CSS en el header

La siguiente forma, un poco mas sofisticada, es escribir los estilos dentro de uno, o más, elementos "style".

Para eso tenemos que saber como "linquear" las reglas CSS que queremos aplicar con los elementos del body.. En otra tarjeta vemos sobre los selectores css.

Es una buena practica poner todos los estilos en un unico elemento en el head, pero pueden tambien estar en el body.
